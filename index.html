One-time Firebase setup

📋 Setup steps

1Go to console.firebase.google.com and sign in with Google.
2Click Add project → name it kitchen-inventory → disable Analytics → Create project.
3In the sidebar click Firestore → Create database → Start in test mode → choose europe-west2 → Enable.
4On project overview click the </> web icon → nickname kitchen-app → Register app.
5Copy the four values from the config block into the boxes below.
